# Расчет экономики ИИ

## Проект Telegram бот технической поддержки платежной системы

Вероятно, вопросы в ТП платежной системы могут разделяться на два вида:
* Фиксированные вопросы касательно финансовых операций (получить статус, номер, чек) 
* Вопросы с подключением по АПИ, глубокие проверки по логам и процессу финансовых операций, предварительные обсуждения договоров и пр.

### Фиксированные вопросы (~70% трафика)
Данный вид вопросов можно легко сформировать в FAQ, особенно на основе ТГ бота. Что явно сэкономит финансовую составляющую проекта и даст сразу четкий понятный ответ сразу подогнанный под удобный для чтения формат.
Такой вид работы явно не потребует существенных финансовых затрат и может быть легко реализован в рамках одного сервиса для бота.

**Примеры запросов:**
- статус транзакции / платежа
- номер и реквизиты операции
- запрос чека или выписки
- информация по лимитам и тарифам
- стандартные вопросы по документации

### Вопросы с привлечением ИИ (~30% трафика)
Работу с ИИ также можно формально разделить на два блока: 
* Упрощенные вопросы - вопросы из разряда разъяснить документацию, пояснить FAQ более детально, дать информацию по лимитам и пр. что не удалось сделать жестко в первом разделе.  
* Вопросы требующие глубокого анализа - решение спорных вопросов, глубокий анализ логов операций, решения по другим нестандартным вопросам. 


#### Упрощенные вопросы

Обрабатываются бюджетной моделью — **DeepSeek V3.2**.

| Параметр           | Значение                           |
|--------------------|------------------------------------|
| Доля от AI-трафика | ~70% AI-запросов (~21% от общего)  |
| Запросов в месяц   | ~9 450                             |
| Модель             | DeepSeek V3.2                      |
| Цена API           | $0.28 in / $0.42 out за 1M токенов |
| Стоимость in       | 7.56M × $0.28 ≈ $2.12 (~159 ₽)     |
| Стоимость out      | 5.67M × $0.42 ≈ $2.38 (~179 ₽)     |
| **Итого в месяц**  | **~$4.50 (~338 ₽)**                |

#### Вопросы требующие глубокого анализа

Обрабатываются качественной моделью — **Claude Sonnet 4.6**.

| Параметр           | Значение                            |
|--------------------|-------------------------------------|
| Доля от AI-трафика | ~30% AI-запросов (~9% от общего)    |
| Запросов в месяц   | ~4 050                              |
| Модель             | Claude Sonnet 4.6                   |
| Цена API           | $3.00 in / $15.00 out за 1M токенов |
| Стоимость in       | 3.24M × $3.00 ≈ $9.72 (~729 ₽)      |
| Стоимость out      | 2.43M × $15.00 ≈ $36.45 (~2 734 ₽)  |
| **Итого в месяц**  | **~$46.17 (~3 463 ₽)**              |


## Сравнение стратегий

| Стратегия              | Модели                | Стоимость / мес |
|------------------------|-----------------------|-----------------|
| Только DeepSeek V3.2   | Бюджет                | ~567 ₽          |
| **Гибрид 70/30**       | **DeepSeek + Sonnet** | **~12 600 ₽**   |
| Только Sonnet 4.6      | Средний               | ~38 475 ₽       |
| Только Claude Opus 4.6 | Флагман               | ~64 125 ₽       |

> Примечание: гибрид считается от полного AI-трафика (30% объёма).
> При исключении 70% фиксированных вопросов реальная стоимость AI-части —
> ~**3 801 ₽/мес** (338 + 3 463 ₽).

### Локальное развёртывание (Selectel)

| Вариант            | Состав                           | Стоимость / мес |
|--------------------|----------------------------------|-----------------|
| Аренда Cloud GPU   | RTX 4090 прерываемая + DevOps    | ~29 810 ₽       |
| Собственный сервер | Амортизация + колокация + DevOps | ~34 667 ₽       |

Капитальные затраты на собственный сервер: ~**280 000 ₽** (RTX 4090 + сервер, амортизация 24 мес).

**Точка безубыточности:** собственный сервер vs чистый Sonnet — **не окупается** (38 475 ₽/мес < 34 667 ₽/мес разница слишком мала: 280 000 / 3 808 ≈ 74 мес).
Против гибридной стратегии — **не окупается тем более** (сервер дороже на всём 36-месячном горизонте).

---

### Обоснование выбора стратегии

**Рекомендуемая стратегия: гибридное облако с трёхуровневым роутингом.**

1. **FAQ / жёсткий роутинг (70% трафика)** — нулевые AI-затраты, мгновенный ответ.
2. **DeepSeek V3.2 (21% трафика)** — упрощённые AI-вопросы, ~338 ₽/мес.
3. **Claude Sonnet 4.6 (9% трафика)** — сложные аналитические запросы, ~3 463 ₽/мес.

Итоговая реальная стоимость AI-компоненты: **~3 801 ₽/мес** (~$50.67).

Локальное развёртывание на Selectel нецелесообразно при данном объёме: стоимость аренды GPU
(29 810 ₽/мес) в 7× превышает гибридное облако, а собственный сервер (34 667 ₽/мес) не окупается
даже за 6 лет. Переход на локальную инфраструктуру имеет смысл при росте нагрузки
свыше **5 000 запросов в день**.

P.S. Цифры представлены из расчета апреля 2026г.

P.S.S. Частично для расчетов был использован Claude Sonnet что бы сократить затраты времени на расчеты. Выбор стратегии собственный.
